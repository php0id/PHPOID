Class AMI_Tx_Cmd_Storage

Abstract storage transaction command.

Expects in $oArgs constructor arguments: - mode - command execustion mode (AMI_iTx_Cmd::MODE_* constants); - source - source file path; - target - target file path; - content - (optional) content to set by AMI_Tx_Cmd_FS::set().
